

 									FUNCTIONS IN PYTHON — TOPIC-WISE FLOWCHART



## 1️⃣ FUNCTION BASICS


START
  ↓
Need reusable code?
  ↓
YES
  ↓
Define Function (def)
  ↓
Give Name + Parameters
  ↓
Write Function Body
  ↓
(Optional) return value
  ↓
Call Function
  ↓
END
```



## 2️⃣ FUNCTION SYNTAX


def function_name(parameters):
    statements
    return value

Flow:
Define → Store in memory → Call → Execute → Return
```



## 3️⃣ TYPES OF FUNCTIONS


                 Functions
                     ↓
     -------------------------------------
     ↓           ↓            ↓           ↓
 Built-in    User-defined   Lambda   Recursive


* Built-in → `print()`, `len()`
* User-defined → created using `def`
* Lambda → one-line function
* Recursive → function calling itself



## 4️⃣ PARAMETERS & ARGUMENTS


             Parameters
                 ↓
   -----------------------------------
   ↓         ↓         ↓         ↓
Positional  Keyword  Default   Variable

Flow:

Call Function
   ↓
Arguments Passed
   ↓
Match with Parameters
   ↓
Execute Function
```

## 7️⃣ VARIABLE SCOPE


             Scope
               ↓
      -------------------
      ↓                 ↓
   Local             Global
      ↓                 ↓
Inside function     Outside function
```

Flow:

Call Function
   ↓
Check variable
   ↓
Local → use local
Else → use global

## ✅ Quick Revision Points

* `def` → define function
* Call → executes function
* `return` → gives output
* Parameters = variables
* Arguments = actual values
* Scope = visibility of variables
