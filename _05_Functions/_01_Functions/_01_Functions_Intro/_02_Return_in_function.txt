 The 'INPUTS' here are called "PARAMETERS"  and the 'OUTPUT' is called "RETURN"

Example: 
def sum(a, b):   -----> This function will always ask for the value of 'a' and 'b' == PARAMETERS === 'a' & 'b' are just placeholders here
	s = a + b  ---> here 's' is just a name of operation, it could be anything
	return s  ----> 'RETURN' is the output of the named function (here it is 's')
