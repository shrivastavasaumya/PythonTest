 
  Small anonymous function defined or represented by 'lambda'

  Example:   lambda a,b,c : a+b+c 
                              |------- (this is an expression)
                            
    # Here we can have multiple parameters in this function but CAN ONLY HAVE ONE EXPRESSION 
    Can use Ternary Operators
    Drawbacks == Can't be used for {complex tasks / multiple condition function / Loops/ multiple variable}

    # Can be helpful for small tasks {Multiplication/ division/averages/addition/subtraction}

   { SYNTAX:             }
   { lambda a,b: a+b     }

   Example to use this with a variable  ---------   

   avg = lambda a,b : (a+b)/2
   print(avg(4,5))      ======= 4.5 


   USAGE OF LAMBDA FUNCTION

   1) In higher order functions, that is filter(), map(), sorted() and reduce() 
  (highter order functions are those functions which takes another function as argument or returns a function from itself)



sorted() = use key

students = [('charlie', 78) , ('alice', 85), ('bob', 90)]
print(sorted(students, key = lambda x: x[1]))
                                                                    {sorting on the basis of marks}
[('charlie', 78), ('alice', 85), ('bob', 90)]


students = [('charlie', 78) , ('alice', 85), ('bob', 90)]
print(sorted(students))
                                                                    {sorting on the basis of name}
[('alice', 85), ('bob', 90), ('charlie', 78)]
