 
                                                                     Truthy and Falsy

        Truthy is equivalent to True                                          Python considers everything as 'Truthy'or 'Falsy'
        Falsy is equivalent to False 
                                               Example: print(bool(1))              -->    True
                                                        print(bool(0))                     False
                                                        print(bool("effdyjdjh"))           True
                                                        print(bool())                      False

    # If we will write ANYTHING in boolean, it will give the output as --->  True

# Falsy values in python
                          print(bool(0))
                          print(bool(0.0))
                          print(bool(''))       --- (Empty string)
                          print(bool({}))       --- (Empty dictionary)
                          print(bool([]))       --- (Empty list)
                          print(bool(None))
# Truthy values in python
                          print(bool(42))        
                          print(bool(-1))
                          print(bool('hello'))
                          print(bool([1, 2]))
