                                                              TO REPLACE     

string.replace(old, new, count)--- syntax
txt = 'Hello World'
print(txt.replace('World','Python'))-----> Hello Python
print(txt.replace('world','Python'))-----> Hello World (No change , because incorrect casing of 'w' and 'W')

[NOTE : Python is case sensitive, so be careful of casing and characters]


The role of ' COUNT ' in the syntax of 'REPLACE' is
Example- txt = 'banana, banana, banana'  ------ (want to replace 2 of them with 'apple')
         print(txt.replace('banana', 'apple', 2)) ----- apple apple banana
         print(txt.replace('banana', 'apple')) ----- apple apple apple (To replace all)
         print(txt.replace('banana', 'apple', 0)) ----- banana, banana, banana ( no change because of '0')


                                                              TO CHECK THE NATURE OF THE TEXT
                                                              
TEXT IS FULLY ALPHABETICAL OR NOT ----> print(text.isalpha()) 
                        Example- text1 = 'PYTHON'
                                 text2 = 'PYTHON3'       --------------> print(text1.isalpha())    ------> True
                                                                         print(text2.isalpha())    ------> False (Because it has a number also)

TEXT IS FULLY NUMERICAL OR NOT ----> print(text.isdigit())                                                                    
                        Example- text1 = '2322'
                                 text2 = '2python'       --------------> print(text1.isdigit())    ------> True
                                                                         print(text2.isdigit())    ------> False (Because it has alphabets also)

TEXT IS ALPHA-NUMERIC OR NOT ----> print(text.isalnum())                                    [NOTE:THESE EMPTY BRACKETS ARE THERE IN THE FUNCTION TO CALL]         
  (LETTERS AND NUMBERS)                                                           
                        Example- text1 = '2322'
                                 text2 = '2python'      --------------> print(text1.isalnum())    ------> True
                                                                        print(text2.isalnum())    ------> True      
                But, if -------> text3 = '2 python      --------------> print(text3.isalnum())    ------> False (Because alphanumeric includes numbers and alphabets only
                                                                                               and does not consider 'SPACES' given in the string as alphanumeric so --- False)

                                                                  STARTS WITH / ENDS WITH


                                                  Example:    text = 'Python is amazing'
                                                
Does the string starts with a certain mentioned character ---> print(text.startswith('Python')) ---> True

Does the string ends with a certain mentioned character --->  print(text.endswith('is')) ---> False

Does the string starts / ends with a certain mentioned character with a definite start and end ---> print(text.endswith('is',0,9)) --->('9' is ending index = exclusive)
