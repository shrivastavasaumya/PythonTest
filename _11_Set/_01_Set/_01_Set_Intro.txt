
                                                       Sets
                                                     ________

                 Built-in data type which stores multiple items just like a list and tuple 

	A)	Unlike list sets are - Unordered  
	B)  Unindexed - can't get things using index
	C)  Mutable 
    D)  Uniqueness of element collection of (without repetition) they don't allow duplicates
	E)  uses - {} curly braces
	F)  Heterogenous things can be stored (string, integer, float, boolean)
	G)  Should have immutable items in a set (cannot have list in it cause it is mutable)
        Example - mixed = {1, "hello", 3.14, True, (1, 2, 3), [1, 2, 3]}
															   ---|------->> this is a list, have to remove it
				  mixed = {1, "hello", 3.14, True, (1, 2, 3)}
