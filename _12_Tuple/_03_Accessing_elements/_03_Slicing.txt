3️.) Slicing

~ Get a range of elements

t = (10, 20, 30, 40, 50)

print(t[0:2])   # (10, 20)
print(t[1:4])   # (20, 30, 40)

=> Rule:

t[start : end]
Start → included 
End → excluded 

=> So:

t[0:2] → index 0 to 1 → (10, 20)

* More Slicing Examples
print(t[:3])    # (10, 20, 30)
print(t[2:])    # (30, 40, 50)
print(t[-3:-1]) # (30, 40)
