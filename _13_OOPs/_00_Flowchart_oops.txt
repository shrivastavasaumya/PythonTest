OOP (Object-Oriented Programming)
│
├── Class (Blueprint)
│   │
│   ├── Definition (class keyword)
│   ├── Attributes (Variables)
│   │   ├── Instance Variables
│   │   └── Class Variables
│   │
│   ├── Methods (Functions inside class)
│   │   ├── Instance Methods
│   │   ├── Class Methods (@classmethod)
│   │   └── Static Methods (@staticmethod)
│   │
│   ├── Constructor
│   │   └── __init__()
│   │
│   └── Special (Magic/Dunder) Methods
│       └── __str__(), __repr__(), etc.
│
├── Object (Instance of Class)
│   │
│   ├── Creation (object = ClassName())
│   ├── Access
│   │   ├── object.variable
│   │   └── object.method()
│   │
│   └── Memory Concept
│       └── Each object has separate data
│
├── self Keyword
│   └── Refers to current object
│
├── Encapsulation (Data Hiding)
│   ├── Public
│   ├── Protected (_var)
│   └── Private (__var)
│
├── Inheritance (Reusability)
│   ├── Single
│   ├── Multiple
│   ├── Multilevel
│   └── Hierarchical
│
├── Polymorphism (Many Forms)
│   ├── Method Overloading (conceptual in Python)
│   └── Method Overriding
│
└── Abstraction (Hiding Implementation)
    ├── Abstract Classes
    └── Abstract Methods
