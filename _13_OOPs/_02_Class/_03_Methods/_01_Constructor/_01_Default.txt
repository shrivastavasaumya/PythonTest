1. Default Method (No Parameters)

A default method is a method that does not take extra arguments (only self).

# Syntax
class Demo:
    def show(self):   # default method
        print("Hello, this is a default method")

obj = Demo()
obj.show()

~ Key Point
Only self is present
Works with fixed behavior
No input from user


| Feature     | Default Method | Parameterized Method     |
| ----------- | -------------- | ------------------------ |
| Parameters  | Only `self`    | `self + extra arguments` |
| Input       | No             | Yes                      |
| Flexibility | Fixed          | Dynamic                  |
| Example     | `show()`       | `show(name)`             |
