
                                                INHETRITANCE -

Reusing attributes and methods from a parent /(base) class
and to the one we give it to use (child class) /derived class

class employee:      -->   parent class
    start_time = '10 am'      
    end_time = '6 pm'

class teacher(employee):   --> to call parent class attributes we have to call it in a parenthesis
    def __init__(self, subject):
        self.subject = subject

t1 = teacher("hindi")

print(t1.subject, t1.start_time, t1.end_time)     # hindi 10 am 6 pm
