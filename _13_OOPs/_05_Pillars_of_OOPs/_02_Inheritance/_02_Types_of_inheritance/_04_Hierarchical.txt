4. Hierarchical Inheritance

👉 Multiple child classes inherit from one parent class.

class Parent:
    def show(self):
        print("Parent class")

class Child1(Parent):
    pass

class Child2(Parent):
    pass

obj1 = Child1()
obj2 = Child2()

obj1.show()
obj2.show()                 # Output - Parent class
                                       Parent class
 


✅ Flow:

        Parent
       /      \
   Child1   Child2
